# ForgeImpact

AI Multi-Agent System for Engineering Change Impact Analysis in Aerospace & Defense

## Project Overview
A professional multi-agent AI application that helps program managers analyze the cross-functional impacts (cost, schedule, risk, manufacturing) of engineering design changes.

**Tech Stack**: Python, LangGraph, LlamaIndex, Groq, Streamlit, Chroma
