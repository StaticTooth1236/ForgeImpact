**1. Introduction**

This document outlines the key specifications for the Eurus Systems MAAP-1 program, as outlined in the Program Requirements Baseline (PRB). The MAAP-1 program aims to design, develop, and deploy a microwave atmospheric absorption platform that provides high-resolution, real-time atmospheric sensing and communication capabilities.

**2. System Overview**

The MAAP-1 platform is designed to operate in the 24 GHz frequency band, with a minimum data rate of 100 Mbps and maximum latency of 10 ms. The platform consists of the following key components:

* Transceiver unit
* Antenna system
* Data processing unit
* Power supply and cooling system
* Installation and operation interface
